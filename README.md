# Piller Metal — Sistema editável estático

Projeto sem build obrigatório. Pode hospedar direto na Vercel, Netlify ou Netlify Drop.

## Como publicar no Netlify Drop
Arraste a pasta inteira `piller_editavel` ou o conteúdo dela.

## Como publicar na Vercel
Suba a pasta no GitHub e importe na Vercel como projeto estático. Não precisa de comando de build.

## Como editar
Abra o site e clique em `Admin`. As alterações ficam salvas no navegador via `localStorage`.
Use `Exportar JSON` para salvar a configuração e depois substituir o `mock-database.json` no código.

## Recursos incluídos
- Tema base parecido com o original.
- Produtos por categoria.
- Cadastro, edição, exclusão e ordenação de produtos.
- Textos editáveis.
- Blocos HTML sanitizados.
- Botões de direcionamento configuráveis.
- Modo layout com arraste dos botões do hero.
- Popups próprios de produto e representantes.
- Biblioteca com imagens extraídas/referenciadas do material original.
